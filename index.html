Создание нового репозитория: Импорт с ПК
  1. Заходим в папку с файлами сайта (расположенных на нашем ПК): cd c:/project/наименование-папки
  2. git init - инициализация пустого/не пустого репозитория
  3. git status - запросить статус файлов
  4. git add . - добавляем файлы
  5. git commit -m "comment" - добавляем текущий "снимок"
  6. При создании нового репозитория в gitHub смотрим на раздел:
    …or push an existing repository from the command line
  7. git remote add origin https://github.com/kuznetsovandrey76/название-репозитория.git - привязываем репозиторий с git'a
  8. git push -u origin master - после этой команды файлы переходят git
