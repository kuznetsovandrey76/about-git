Open GitBash
Во время первого захода сразу задаем конфигурации
git config --global user.name "User Name"
git config --global user.email "user@email.ru"

All setting git config --list

Создание нового репозитория:
  1. Main
  2. Заходим в папку с файлами сайта (на нашем ПК): cd c:/project/folder
  3. git init - инициализация репозитория
  4. git status - запросить статус файлов
  5. git add . - добавляем файлы
  6. git commit -m "comment" - добавляем текущий "снимок"
  7. git commit -a -m "comment" - добавление последующих "снимок" без команды "git add"
  8. Push
    9. git remote add origin https://github.com/kuznetsovandrey76/название-репозитория.git - привязываем репозиторий с git'a
  9. git push -u origin master - после этой команды файлы переходят на github
  10. git push - последующий перенос на github
  11. Branch
  12. git branch new-name-branch - создание ветки без перехода в нее
  13. git checkout -b new-name-branch - создание ветки с переходом в нее
  14. git checkout name-of-branch - переключение между ветками
